# Water Balloon Battle

Dive into the fun with Water Balloon Battle, a simple browser game where you can throw and dodge water balloons! Perfect for quick entertainment for youngsters and old fogies alike.

- Play on: any compatible browser
- Mobile/tablet/desktop play is fully supported
- Touch screen compatible
